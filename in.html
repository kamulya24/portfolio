Pythonic Coffee Machine Simulator

This Python script simulates a coffee machine, managing resources like water, milk, coffee beans, cups, and money. Users can choose from espresso, latte, or cappuccino, with methods to check resource availability and purchase coffee. It prompts users for their choice, verifies resource availability, calculates payment, and dispenses coffee with change if applicable. The main function orchestrates the interaction, looping until the user exits. This script encapsulates a basic coffee machine simulation with intuitive user interaction and resource management.
